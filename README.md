# AI Dungeon Master – Adaptive Narrative Engine

A modular Python-based AI storytelling engine featuring class-specific story arcs, adaptive combat systems, and branching narrative logic.

## Overview

AI Dungeon Master is a command-line RPG engine where players choose a character class and progress through dynamically evolving story paths. The system adapts based on player decisions and character attributes.

## Features

- Object-oriented modular architecture
- Class-specific branching narratives
- Adaptive combat system
- Inventory management system
- Decision-tree based story progression
- Dynamic difficulty scaling

## Tech Stack

- Python
- Object-Oriented Programming
- Decision Trees
- Modular Game Architecture

## Project Structure

- `main.py` – Game loop
- `character.py` – Character classes
- `world.py` – World management
- `combat.py` – Combat logic
- `story.py` – Narrative engine
- `inventory.py` – Inventory system
